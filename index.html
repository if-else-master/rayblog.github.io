Project

克隆音頻非同步影片翻譯軟體 →

#開源 #T-School #國際交流 #OpenAI
在數位實驗高中，我們致力於推動國際化教育，經常與其他國家的學校進行交流與合作。為了更好地分享我們的成果，例如非同步課程與社團課程，我們希望製作高品質的國際教育交流影片。然而，在實際分享的過程中，語言障礙成為一個主要挑戰，因為不同國家的觀眾可能無法理解影片中的中文內容。 為了解決這個問題，我計畫開發一個軟體，能將影片中的中文內容翻譯成16種語言，並生成對應的語音。與現有的工具如 Google 翻譯相比，我希望進一步提升功能，加入克隆影片中人物音色的技術。例如，如果影片是一堂烹飪課，我希望軟體能保留授課者的音色，將內容翻譯成日文後重新生成日文語音，並結合影片，打造出一段高擬真的日文版烹飪課影片，這樣能更有效地傳遞我們的課程內容，促進國際間的理解與合作。

開放無線網絡的安全風險評估 →

#WI-FI #DNS劫持 #ARP詐騙 #Python
在日常生活中，我們可能會去咖啡廳或K書中心，那我們可能會使用那邊的Wi-Fi，但開方空間的 Wi-Fi通常未使用強加密協議，讓攻擊者容易更改或攔截流量。那這個網路也有可能是一個未授權的AP(流氓熱點)，攻擊者可能架設一台假的免費Wi-Fi，吸引用戶連接，然後執行DNS劫持或ARP欺騙。那我想要用 Python 的 scapy、dnspython、nmap函式庫來檢測ARP欺騙跟DNS劫持，讓我們在使用這種開方空間網路之前可以先檢查一遍。
